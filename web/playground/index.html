Cheat Sheet

          

            Playground は std::prelude と std::undet::* を暗黙 import している。
          

        

        

          

            
              

                Group
                Form
                Notes
              

            
            
              

                値
                1, 2.0, true, "text"
                整数、浮動小数、真偽値、文字列。
              

              

                束縛
                my x = 1 + 2
                ローカルな名前を作る。
              

              

                公開束縛
                our f x = x + 1, pub answer = 42
                our と pub は型推論結果の Types 欄にも出る。説明用の小さな API を見せるときに便利。
              

              

                参照
                my $x = 10, $x, &x = 11
                $ は読み取り、& は書き込み先。普通の変数再代入ではなく、参照の読み書きを明示する。
              

              

                関数
                my id x = x, \x -> x + 1
                名前付き関数とラムダ。
              

              

                block
                ({ my x = 1; x + 1 })
                複数の文をまとめて最後の式を値にする。インデント block と brace group のどちらでも書ける。
              

              

                条件分岐
                if x == 1: "one" else: "other"
                条件で値を選ぶ。
              

              

                パターン
                case xs: [] -> 0; [x,..xs] -> x
                リスト、enum、record を分岐できる。
              

              

                リスト
                [1, 2, 3], [..xs, x]
                リスト literal と spread。
              

              

                リスト関数
                map(xs, f), filter(xs, pred), sort(xs)
                empty, singleton, cons, uncons, fold, rev, first, last もある。
              

              

                加算
                1 + 2, "a" + "b", [1] + [2]
                Add role。int、float、str、list で使える。
              

              

                比較
                x == y, x <= y
                Eq と Ord role。int と float が主な対象。
              

              

                長さ
                [1, 2, 3].len, "abc".len, len xs
                Len role。list と str に実装されている。method と関数呼び出しのどちらでも使える。
              

              

                範囲
                1..3, 1..<3, ..3, 1..
                range 値を作る。list/str index と組み合わせる。有限範囲は Fold でも使える。
              

              

                range fold
                (((0..<5).fold) 0) (\acc i -> acc + i)
                有限範囲を int の列としてたたむ。この例は 0 + 1 + 2 + 3 + 4。
              

              

                index
                xs[0], xs[1..3], s[0], s[1..3]
                Index role。xs.index 0 のような method 呼び出しもできる。
              

              

                opt
                nil, just x
                first, last, uncons, .once などが返す。constructor は prelude からそのまま使える。
              

              

                非決定
                each [1, 2, 3]
                playground では std::undet::* を暗黙 import している。
              

              

                guard
                guard (x == 3)
                std::undet::guard は条件が false のとき fail() する。each と組み合わせて候補を絞る。
              

              

                非決定 handler
                (each [1, 2] + each [4, 5]).list
                .list は全解、.logic は探索順、.once は最初の解。
              

              

                junction
                if all [1, 2, 3] < any [2, 3, 4]: 1 else: 0
                any と all で比較式を分岐させる。
              

              

                record
                { width: 1, height: 2 }
                pattern では { width = 1, ..rest } のように default/spread も使える。
              

              

                省略可能引数
                my area({width = 1, height = 2}) = width * height
                record pattern の default を使う。area { width: 3 } や area {} のように呼べる。
              

              

                struct with
                struct point { x: int } with: our p.double = p.x + p.x
                データ型と method をまとめて定義する。point { x: 3 }.double のように呼べる。
              

              

                enum / case
                enum color = red | blue, case c: color::red -> 1
                constructor pattern で分岐する。payload つき variant も扱える。
              

              

                role
                role Display 'a: our a.show: str
                impl Display int: our x.show = ... のように method を足す。
              

              

                effect
                act io: our read: () -> int
                catch io::read(): io::read(), k -> k 1 のように捕まえる。
              

              

                catch
                catch action: effect::op(x), k -> k x
                effect operation と継続 k を受け取って処理する。捕まえない effect は外側へ残る。
              

              

                do
                my x = f do
                Haskell の do 記法ではなく、同じ block の後続計算を差し込む marker。標準ライブラリの制御関数で使う。
              

              

                for control
                for x in xs: if x == 2: next else: ()
                for は内部で std::flow::loop::for_in を使う。last は break、next は continue、redo は同じ要素をやり直す。
              

              

                sub return
                sub: for x in 0..: if x == 5: return x else: ()
                sub: は prefix の return x を捕まえて値 x にする。無限範囲や fold の途中脱出を値として扱える。
              

            
          

        

      

      

        

          
Language Guide

          

            Yulang は型推論、role、代数的 effect を中心にした実験的な言語です。
            この playground で試すときに特に重要な見方を短くまとめています。
          

        

        

          

            
読む順番

            

              まずは左の editor で小さな式を書き、Result と Types を見ます。
              our や pub の binding は Types に推論結果が表示されます。
            

            
our twice x = x + x
twice 21

          

          

            
型と role

            

              + や <= は固定の組み込み演算ではなく、prelude の role method として解決されます。
              多相な binding では Add<α> のような requirement が型に残ります。
            

            
our twice x = x + x
// twice: Add<α> => α -> α

          

          

            
制御は effect で見える

            

              sub:、return、for、last は、標準ライブラリの effect handler と専用の lowering がつながって動きます。
              Examples の Effects では、unit -> [console] int のような型と handler 後の型を見られます。
            

            
sub:
    for x in 0..:
        if x == 5: return x
        else: ()
    0

          

          

            
参照は明示的

            

              $x は read、&x = value は write です。
              通常の変数再代入ではなく、局所的な effect scope として実装されています。
            

            
{
    my $x = 10
    &x = $x + 1
    $x
}

          

          

            
非決定性

            

              each は候補を選び、.list や .once が計算を実行します。
              playground では std::undet::* が暗黙に使えます。
            

            
(each [1, 2, 3] + each [4, 5, 6]).list

          Short English summary
          

            Yulang is an experimental expression-oriented language built around type inference,
            structural types, roles, and algebraic effects. Many control-flow features are
            expressed by the standard library, while syntax such as sub:,
            for, $x, and effectful if conditions connect
            user code to those library-defined handlers.
