# Archived Notes

This directory keeps notes that are no longer part of the active working set.
They are preserved for historical context and for reconstructing why older
implementation choices were made.

Current planning and specifications live in:

- `tasks/current.md`
- `notes/todo/`
- `notes/design/`
- `spec/`

Archived notes are not active instructions. If an old note contains a useful
rule, promote the rule back into `spec/`, `tasks/current.md`, or a current
`notes/todo/` file before relying on it.
