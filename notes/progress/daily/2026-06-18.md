## infer small optimization loop

- `instantiate_use` で `Def::Let.scheme` を丸ごと clone してから instantiate していたため、
  `poly.defs` 上の scheme を直接参照するようにした。
  `analysis.instantiate_clone_scheme` の内訳には出ない外側 clone なので、wall/infer 全体で見る。
- `quantify_component` の finalize loop で def parent map を def ごとに作り直していたため、
  component ごとに一回だけ作るようにした。`q_fin` は 13ms 前後から 5ms 前後へ低下。
- role resolve prepass は reachable role が空でも `compact` を clone していたため、空なら `None` で返し、
  resolver を呼ばないようにした。効果は小さめで、`g_rrole` が少し下がる程度。
- constraint machine は `Pos::Bot <: _` / `_ <: Neg::Top` を enqueue 後の `step_subtype` で no-op にしていたため、
  enqueue 前に捨てるようにした。`showcase` では `constraint.work_items` が 42159 から 37994 へ低下。
