# 2026-05-19

- Added an MMTk allocation profile for functional-language GC tuning probes.
  `MmtkHeap` now records total object count, trace slots, raw payload bytes,
  total object bytes, max trace slots, max raw payload bytes, object-kind
  buckets, and compact storage-shape buckets as allocations happen.
- Generated MMTk executable harnesses can dump the profile with
  `YULANG_MMTK_CPS_HEAP_STATS=1`. The lines are prefixed with
  `[JIT-MMTK-STATS]` and are no-ops unless the env var is set.
- First 20x20 readings:
  - `.list.len`: heap objects=1,026, trace_slots=4,998; list dominates with
    899 objects and 4,872 trace slots.
  - `.list.say`: heap objects=12,459, trace_slots=58,165; string has 5,984
    objects and list has 4,748 objects, with list carrying 46,903 trace slots.
